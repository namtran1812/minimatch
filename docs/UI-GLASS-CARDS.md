# Unified Light Glass Cards

The market-summary cards use the same light pastel glass treatment as the global session metrics.

- Best bid: blue-white gradient with green value
- Midpoint/spread: lilac-white gradient with neutral data text
- Best ask: pink-white gradient with red value
- Shared border, radius, spacing, shadow, typography, and tabular number formatting

This avoids isolated dark blocks and keeps exchange summaries visually consistent with Submitted, Accepted, Rejected, Trades, Volume, Active Orders, Spread, and Imbalance.
