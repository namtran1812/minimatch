# Frontend Design System

MiniMatch uses a restrained Y2K-inspired visual identity around a professional trading interface.

## Typography

Only two font families are used:

1. **Interface:** Avenir Next / Helvetica Neue / Arial
2. **Market data:** SF Mono / Menlo / Monaco / Consolas

The interface follows a fixed type scale:

- 48 px — product title
- 24 px — key metric values and workspace headings
- 16 px — section headings
- 14 px — body text and form controls
- 12 px — supporting text and table values
- 11 px — labels and metadata

## Layout

- Four summary metrics per row on desktop.
- The command deck spans the full workspace width.
- The order book and activity feed form the primary two-column workspace.
- Analytical pages use at most two panels per row.
- Numeric values use tabular figures and right-aligned columns.

## Visual language

Pastel blue, pink, lilac, and mint surfaces provide personality. Decorations stay outside data-dense regions. Tables, charts, forms, and operational messages remain precise and professional.
