How to demonstrate the system
  1. Load FIFO match and show that older orders execute first at the same price.
  2. Load FOK / post-only and inspect rejected versus filled execution reports.
  3. Start automated flow, then open Microstructure to watch spread, imbalance, microprice, and depth evolve.
  4. Switch symbols after loading Multi-symbol to prove independent books and deterministic routing.

  2. Run source scripts/setup_massive.sh.
  3. List or download an entitled Flat File.
