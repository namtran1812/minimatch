Matching Engine

A deterministic, price-time-priority exchange core with explicit order lifecycle semantics and reproducible execution.

  │
  ├── validate
  ├── assign sequence
  ├── cross opposing liquidity
  ├── generate fills
  └── rest remaining quantity

  │
  ├── locate order
  ├── unlink from price level
  └── emit cancellation result

  • Limit

Matching guarantees

  • FIFO within each price level
  • Best-price selection
  • Partial-fill support

Engineering focus

  • Cache-conscious data structures
  • Stable order references

ENGINEERING JOURNAL / 09

Build Log

MiniMatch was developed incrementally: correctness first, then protocols, market connectivity, routing, replay, execution, observability, and measurable performance.

01
foundation

Deterministic matching core

Implemented price-time priority, limit and market orders, IOC behavior, cancellation, partial fills, and execution reports.

02
systems boundary

Binary gateway and event log

Added protocol encoding, TCP transport, replayable events, snapshots, and deterministic recovery tools.

03
market connectivity

Live multi-venue L2

Built adapters for Coinbase, Binance, and Kraken, including snapshots, incremental updates, sequence checks, and checksum handling.

04
execution intelligence

Consolidation and routing

Combined independent venue books and implemented deterministic multi-leg routing with fees and liquidity-aware execution plans.

05
lifecycle

Execution, risk, persistence, and FIX

Added execution algorithms, parent-child order tracking, pre-trade controls, SQLite persistence, FIX sessions, and gateway translation.

06
reproducibility

Binary recording and interactive replay

Captured live sessions and built deterministic replay with play, pause, restart, speed, and seek controls.

07
operations

Health and latency instrumentation

Added venue health, counters, quote age, bounded percentile tracking, route timing, serialization timing, and per-feed apply metrics.

08
deployment

Docker, CI, and observability

Added container builds, automated tests, Prometheus integration, Grafana dashboards, benchmark artifacts, and reproducible startup scripts.
